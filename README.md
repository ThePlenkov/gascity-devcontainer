# Devcontainer Features

Collection of modular, reusable devcontainer features for AI development.

## Features

- **gascity** - Gas City AI agent orchestration platform
- **devin** - Devin CLI for AI-powered development
- **claude** - Anthropic Claude configuration

## Usage

```json
{
  "features": {
    "./src/gascity": {},
    "./src/devin": {},
    "./src/claude": {}
  }
}
```

Each feature is independent and can be used in any project, not just Gas City.
